Enablers

You can use the following options to interact with the TPS. It works with the query string. Like so: https://thinking-people-system.house?text=hide.

  • background color: change the primary color in green
  • font color: change the font color in orange
  • display: display without any HUD with the option house-only
  • focus: focus only a part of the TPS. Can be multiple. (Here are all the possible values: )
  • text: hide text to only have the house
